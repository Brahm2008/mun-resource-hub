# MUN Resource Hub

MUN Resource Hub is a beginner-friendly multi-page website designed to help students understand and prepare for Model United Nations (MUN) conferences.

## Features

- Multi-page website structure
- Introduction to MUN basics
- Committee overview section
- Resources and preparation tips
- Practice section with:
  - Random MUN prompt generator
  - Self-score generator for delegate practice

## Built With

- HTML
- CSS
- JavaScript

## Purpose

This project was built as a beginner web development project to practice:
- structuring multi-page websites
- styling with CSS
- adding basic interactivity with JavaScript
- organizing a project for GitHub and portfolio use

## Future Improvements

- Better responsive layout
- More committee-specific resources
- Expanded practice simulations
- Improved scoring logic and delegate feedback
